# US COVID-19 Maps (2020)

## Project Overview
This project presents two interactive web maps showing COVID-19 data in the United States for the year 2020 at the county level. The maps are built using Mapbox GL JS and focus on both COVID-19 case rates and total case counts.

## Maps
- **Map 1 – COVID-19 Case Rates (Choropleth Map)**  
  Shows COVID-19 case rates per 1,000 residents by county.  
  https://irving0604.github.io/us-covid-19-maps/map1.html

- **Map 2 – COVID-19 Case Counts (Proportional Symbols Map)**  
  Shows total COVID-19 case counts using proportional circle symbols.  
  https://irving0604.github.io/us-covid-19-maps/map2.html

## Main Features
- Albers projection for U.S. mapping
- Appropriate basemap for thematic visualization
- Interactive elements (clickable counties and symbols with popups)
- Legends explaining data values
- Titles, descriptions, and data source information

## Libraries Used
- Mapbox GL JS

## Data Sources
- COVID-19 case data: The New York Times
- Population data: 2018 ACS 5-Year Estimates
- County boundaries: U.S. Census Bureau
